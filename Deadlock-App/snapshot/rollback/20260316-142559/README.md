# Deadlock Competitive Intelligence Toolkit

Local deterministic data toolkit for Deadlock using only `https://api.deadlock-api.com`.

## Quick start

```bat
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python src/main.py all
```

## Commands

- `python src/main.py api`
- `python src/main.py dataset`
- `python src/main.py synergy`
- `python src/main.py counters`
- `python src/main.py meta`
- `python src/main.py predict`
- `python src/main.py analyze`
- `python src/main.py export`
- `python src/main.py snapshot`
- `python src/main.py all`
